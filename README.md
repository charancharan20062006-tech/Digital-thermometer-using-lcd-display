# Digital-thermometer-using-lcd-display
Temperature sensor using arduino and display with lcd
Project Report: Temperature Sensor Interfacing with LCD Display Using Arduino
Project Overview

I developed a real-time temperature monitoring system using an Arduino microcontroller, a temperature sensor (LM35), and a 16×2 LCD display. The complete circuit design, simulation, and testing were performed using Tinkercad before implementation. The system continuously senses ambient temperature and displays the measured value on the LCD screen in real time.

Objectives
To interface a temperature sensor with an Arduino board.
To acquire and process sensor data using Arduino programming.
To display real-time temperature readings on an LCD display.
To validate the design through simulation using Tinkercad.
Technical Implementation
Designed and simulated the circuit in Tinkercad.
Connected the LM35 temperature sensor to the Arduino Uno.
Developed Arduino code to read analog sensor values and convert them into temperature readings.
Interfaced a 16×2 LCD module to display temperature data.
Tested and verified the accuracy of readings through simulation.
Skills Demonstrated
Arduino Programming (Embedded C/C++)
Sensor Interfacing and Calibration
Analog-to-Digital Conversion (ADC)
LCD Display Interfacing
Circuit Design and Simulation using Tinkercad
Hardware-Software Integration
Troubleshooting and Testing
Applications

The system can be applied in industrial monitoring, smart homes, environmental monitoring, laboratories, HVAC systems, and automation projects.

Conclusion

This project successfully demonstrates the integration of sensors, Arduino programming, and display modules for real-time monitoring applications. It enhanced my practical understanding of embedded systems, circuit simulation, and hardware-software interaction while providing a scalable foundation for future IoT-based monitoring solutions.
